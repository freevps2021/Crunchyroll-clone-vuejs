# Crunchyroll clone in VueJs

I made another Crunchyroll clone using VueJS, Bootstrap and and API as a part of a school project.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
